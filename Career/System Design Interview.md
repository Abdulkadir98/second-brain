### 4 Step Framework

1. Understand the problem and establish design scope
	1. Clarify constraints and narrow down the features
	2. List functional and non-functional requirements
	3. Back of the envelope estimation for the load on the system
	4. Should care about data privacy such as GDPR etc
  2.  Establish High-level design and get buy-in
	  1. Identify the APIs
	  2. Data model
	  3. High level system architecture
  3. Deep Dive into the system design
	  1. How will you scale the individual components
	  2. What are the bottlenecks?
4. Wrap-up
	1. Talk about Observability, Monitoring, Security etc
