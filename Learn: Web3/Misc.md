Meta Mask - Is an application that creates/import wallet and lets you make transactions on the Ethereum mainnet and other test blockchains

Etherscan.io - Is a block explorer that - shows transactions for a unique address on the mainnet or even other test block chains

Rinkeby faucet - Application that give free test tokens to allow you to test your applications on the Rinkeby network

Every transaction on the Ethereum Blockchain requires “gas” which is another term for transaction fee. Calculated as gas price * gas limit for the transaction. Whoever wants to perform a transaction (Change the state of the Blockchain) needs to pay this gas fee.
